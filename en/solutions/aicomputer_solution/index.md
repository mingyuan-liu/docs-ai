# AI Computer Solution Overview

## Zenow

**Zenow** is a locally deployed AI knowledge assistant desktop application. All data processing is performed locally to fully protect user privacy and data security. It supports multi-model management, intelligent conversation, knowledge base question answering, voice interaction, and other capabilities.

- [Zenow](zenow.md)

## Yumeet

**Yumeet** is a locally deployed AI meeting assistant desktop application. Audio processing, transcription, translation, and summarization can all be completed locally to improve meeting documentation efficiency while protecting private data. Built on a speech pipeline that includes VAD, ASR, diarization, translation, and TTS, together with LLM-based summarization, it supports real-time recording, offline audio import, meeting archiving, and retrieval.

- [Yumeet](yumeet.md)
