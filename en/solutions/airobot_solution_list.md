# Solution Overview

SpacemiT AI Robot reference solutions are designed for ROS and AI robotics applications, with a focus on validating AI capabilities and accelerating solution deployment in real-world robot scenarios:

- **AI Visual Perception**: object detection, human recognition, gesture recognition, visual following, and intelligent video analysis
- **AI Voice Interaction**: speech recognition, speech synthesis, and natural language control
- **Embodied Intelligence Applications**: robotic arm training and inference, mobile manipulation, and humanoid robot motion control
- **Typical Robot Scenarios**: reference solutions for desktop robots, wheeled robots, robotic arms, humanoid robots, and more

These reference solutions will continue to be updated. For the latest solution list, deployment instructions, and case documentation, please refer to the SpacemiT Robot SDK documentation.
